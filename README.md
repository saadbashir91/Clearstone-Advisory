# Clearstone Advisory

Website for Clearstone Advisory — practice operations for financial advisors and professional services firms.

Live at [clearstoneadvisory.ca](https://clearstoneadvisory.ca)

## Files

- `index.html` — The main site (self-contained, no dependencies)

## How this is hosted

This site is deployed via GitHub Pages with a custom domain. To update the site, edit `index.html` and commit — GitHub Pages will redeploy automatically.
